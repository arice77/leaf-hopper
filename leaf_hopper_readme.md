# Leaf Hopper Game

## Overview
Leaf Hopper is a nature-inspired maze game where you play as a forest spirit hopping between leaves and branches. This game is a thematic variation of the classic PacMan arcade game.

## Theme
- **Character**: A cute glowing forest spirit
- **Collectibles**: Light orbs that pulse with a golden glow
- **Enemies**: Poisonous beetles that roam the forest paths
- **Maze Style**: Organic paths made of vines and branches

## How to Play
1. Use the arrow keys or WASD to move your forest spirit through the maze
2. Collect all the light orbs while avoiding the poisonous beetles
3. If you collect all the orbs, more will appear
4. If you touch a beetle, the game is over

## Controls
- **W / ↑**: Move up
- **A / ←**: Move left
- **S / ↓**: Move down
- **D / →**: Move right

## Running the Game
1. Start a local web server (e.g., `python3 -m http.server`)
2. Open your browser to `http://localhost:8000/leaf_hopper.html`
3. Click "Start Game" to begin
